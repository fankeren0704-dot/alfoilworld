📚 铝箔知识库

涵盖铝箔生产的全流程知识，从原材料到成品应用，由30年铝加工行业经验团队整理。

🏭 生产工艺
  • 铝箔定义：厚度在0.2mm以下的铝及铝合金薄板带材，最小工业化生产厚度为0.004mm（4μm）
  • 双零箔/单零箔：厚度0.004\~0.0065mm为双零箔（如4μm、6μm），0.01\~0.1mm为单零箔（如7μm、9μm），0.1\~0.2mm为厚箔
  • 热轧毛料：铸锭→热轧→冷轧→箔轧，组织均匀、力学性能好，适合高端产品
  • 铸轧毛料：铝液直接铸轧成坯→冷轧→箔轧，流程短、成本低；铸轧温度680\~700℃，速度800\~1100mm/min
  • 粗轧：将0.3\~0.5mm的冷轧坯轧至0.05\~0.07mm，压下量大、轧制力高
  • 中轧：将粗轧料继续减薄至0.025\~0.035mm，控制板形和表面质量
  • 双合：两层箔材合在一起轧制，一次轧出两层极薄铝箔，是双零箔生产的标志性工序
  • 精轧：最终减薄至成品厚度（如6μm、7μm），对表面质量和厚度精度要求最高
  • 退火：消除加工硬化、获得所需力学性能（O态或H态），典型退火温度200\~400℃
  • 分切：将宽幅铝箔按客户要求分切成特定宽度，刀具精度直接影响毛刺和端面质量
  • 典型轧制参数：箔轧速度600\~1500m/min，轧制力2000\~6000kN，总压下率可达98%以上
📋 产品标准
  • 厚度公差-高精级：厚度≤0.006mm ±8%、0.006\~0.03mm ±6%、0.03\~0.1mm ±5%
  • 厚度公差-普通级：厚度≤0.006mm ±10%、0.006\~0.03mm ±8%、0.03\~0.1mm ±7%
  • 针孔数-A级：厚度≤0.006mm ≤500个/m²、0.006\~0.01mm ≤300个/m²、0.01\~0.03mm ≤50个/m²
  • 针孔数-B级：厚度≤0.006mm ≤1000个/m²、0.006\~0.01mm ≤600个/m²、0.01\~0.03mm ≤100个/m²
  • 针孔数-C级：厚度≤0.006mm ≤3000个/m²、0.006\~0.01mm ≤1000个/m²、0.01\~0.03mm ≤300个/m²
  • 常用合金：1235（纯铝）、8011（包装通用）、8079（高延伸）、8006（高强度）、1060/1100（工业用）、3003（散热器用）
  • Fe/Si比控制：8011合金Fe含量0.6\~1.0%、Si 0.5\~0.9%，Fe/Si比控制在2.5\~3之间为宜
  • 新旧牌号对照：1070A=L1、1060=L2、1050A=L3、1100=LB2、1200=LG5、8A06=L6
  • 状态代号对照：O=退火态(M)、H12=Y4、H14=Y2、H16=Y1、H18=Y(全硬)
  • 检测标准：GB/T22638系列涵盖铝箔针孔度、表面润湿张力（达因值）、粘附性等检测方法
🔧 装备与维护
  • 轧机类型：四辊不可逆轧机为当前铝箔轧制主流，工作辊+支撑辊结构保证刚度和精度
  • 国际制造商：ACHENBACH（德国阿亨巴赫）、SMSD（西马克）、IHI（日本石川岛播磨）为三大主流品牌
  • 板形控制：VIDIMON（阿亨巴赫）、OPTIROLL（西马克）、ABB板形辊，实时测量并自动调节
  • 测厚仪：X射线在线测厚仪，测量精度±0.15%，实现AGC自动厚度控制
  • 磨床：德国HERKULES高精度轧辊磨床，圆度偏差≤1μm，表面粗糙度Ra≤0.02μm
  • 退火炉：温度控制精度±3℃，升温速率可控，冷却速率50℃/h连续可调，避免粘伤
  • 过滤系统：SCHNEIDER施耐德板式过滤器，配合硅藻土助滤，保证轧制油清洁度
  • 工作辊：直接与铝箔接触的轧辊，直径小（230\~280mm），表面光洁度要求极高
  • 支撑辊：支撑工作辊、承受轧制力的大型辊，直径大（500\~700mm），刚度高
  • 导辊/展平辊/板形辊：导辊引导箔材走向，展平辊消除褶皱，板形辊实时检测板形
🔬 质量管控
  • 压折/斜角：粗中轧道次折痕45°角 → 增加后张力改善展平效果
  • 开缝：中精轧纵向开裂 → 正确设定后张力，避免张力波动
  • 线条：精轧道次出现细线状光泽差异 → 改善润滑油分配和质量
  • 色差：精轧道次光泽不均匀 → 窄料后更换支撑辊，重新磨削
  • 起鼓：中精轧局部凸出不平 → 检查喷液均匀性、降低油温
  • 斜纹/人字纹：润滑不足导致金属流动不均 → 改善润滑、降低压下量
  • 振痕：油膜周期性变化引发振动 → 改变工艺参数避开共振区
  • 氧化水斑：铝箔与水反应生成氧化斑 → 严格控制轧制油含水量≤0.05%
  • 起皱：厚度0.05mm以下张力不足 → 调整展平辊角度和压力
  • 辊印：异物压入箔面形成凹痕 → 确保料头清洁、定期检查辊面
  • 裂边：边部受拉应力过大 → 放松边部张力（负弯调节）
  • 串层：卷取偏离对中线 → 调整展平辊纠偏装置
  • 松卷/箭头：卷取内松外紧产生箭头状突起 → 保证打底张力和质量
  • 针孔：铝箔最常见缺陷，透光可见微孔 → 与毛料冶金质量直接相关，控制铸轧/热轧坯质量
  • 达因值：电池箔表面润湿张力关键指标，一般≥30mN/m，涂碳产品要求≥33mN/m
🔋 应用领域
  • 食品包装：铝箔阻隔性最佳，透湿度0\~0.46g/m²·24h，兼具隔光、隔热、隔气功能
  • 新能源电池箔：锂离子电池集流体用铝箔（12μm为主），异物质控制≤50mg/30万m²
  • 空调亲水铝箔：空调散热器核心材料，表面涂覆亲水涂层提高换热效率
  • 药用铝箔：PTP泡罩包装用铝箔（通常0.02\~0.025mm），针孔直径必须≤0.3mm
  • 电子电容器：电解电容器用铝箔，电阻值与合金纯度（Al≥99.93%）密切相关
  • 装饰铝箔：镜面/亚光铝箔，表面亮度L值可达27，用于装饰、标牌等领域

📖 铝箔行业发展史

从1910年第一条铝箔轧机诞生，到如今全球700万吨市场规模，百年铝箔产业的演进之路。

🌍 全球铝箔发展

1891年
美国和欧洲手工作坊采用多层锤击法生产小张箔，铝箔生产萌芽
1903年
法国A.Gautschi用二辊轧机平张叠轧法轧得厚0.05mm铝箔
1910年
瑞士R.V.Neher与德国August Schmitz合作，成功轧出成卷窄幅铝箔，铝箔进入工业化生产
1950-70年代
冷轧+箔轧工艺成熟，双零箔（厚度＜0.0065mm）实现量产
2000年后
全球铝箔产能向中国转移，中国成为最大铝箔生产国
2020年代
新能源电池箔需求爆发，铝箔进入高端化、功能化新阶段，全球产量约700万吨

🇨🇳 中国铝箔产业

1932年
瑞士铝Alussise、加铝Alcan、英国铝British Aluminium合资在上海建华铝钢精厂，中国铝箔工业起步
1958-1963年
成都715厂、东轻厂装备国产二辊轧机；1959年西北铝从德国进口铝箔轧机
1979年
东轻厂从Achenbach引进四辊不可逆万能轧机，铝箔生产开始进入现代化
1983年
华北铝引进4台日本神户铝箔轧机及生产技术know-how，奠定中国现代化铝箔生产技术基础
1990年代
厦顺、大亚、渤海、上海闵行、西南铝等一大批现代化铝箔厂建成，中国进入世界铝箔强国行列，产量从1.5万吨增至24万吨
2000年至今
两米级2000m/min高速铝箔轧机集中投产，中国铝箔装备和产能进入绝对世界第一，铝箔大量出口
